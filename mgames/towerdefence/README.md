# 🏰 타워 디펜스 게임

초등학교 3학년 아이와 함께 즐길 수 있는 완전한 타워 디펜스 게임입니다!

## 🎮 게임 특징

### 🌟 새로운 기능들
- **보스 시스템**: 3웨이브마다 강력한 보스 등장
- **스테이지 시스템**: 10웨이브마다 스테이지 클리어
- **상점 시스템**: 코인으로 아이템 구매하여 능력 강화
- **홈 화면**: 전투와 상점을 선택할 수 있는 메인 화면
- **코인 시스템**: 점수를 코인으로 변환하여 아이템 구매

### 🎯 게임 목표
- 적들이 경로를 따라 목적지에 도달하기 전에 타워로 공격하여 처치
- 생명력이 0이 되면 게임 오버
- 10웨이브를 클리어하면 스테이지 완료
- 코인을 모아서 상점에서 아이템 구매

## 🎮 게임 방법

### 홈 화면
- **전투 시작**: 현재 스테이지로 게임 시작
- **상점**: 코인으로 아이템 구매

### 게임 화면
- **타워 설치**: 게임 영역 클릭 (골드 50 필요)
- **일시정지**: 게임 일시 중단
- **다시 시작**: 게임 초기화

### 상점 화면
- **공격력 강화** (2코인): 타워 공격력 +10
- **범위 확장** (3코인): 타워 범위 +20
- **공격 속도** (2코인): 타워 공격 속도 +20%
- **골드 부스터** (1코인): 적 처치 시 골드 +5
- **생명력 증가** (5코인): 생명력 +3

## 🎯 게임 요소

### 기본 시스템
- **💰 골드**: 타워 구매에 사용 (적 처치 시 획득)
- **❤️ 생명력**: 적이 목적지에 도달하면 감소
- **🎯 점수**: 적 처치 시 획득
- **🌊 웨이브**: 점점 어려워지는 적들의 공격
- **🪙 코인**: 점수 10점당 1코인으로 변환

### 보스 시스템
- **3웨이브마다 보스 등장**: 👹 이모지로 표시
- **보스 특징**: 크기가 크고, 체력이 많고, 느리게 이동
- **보스 보상**: 일반 적보다 많은 골드와 점수 제공

### 스테이지 시스템
- **10웨이브 클리어**: 스테이지 완료
- **스테이지 진행**: 난이도가 점진적으로 증가
- **점수 변환**: 스테이지 완료 시 점수를 코인으로 변환

## 🎯 게임 특징

### 아이 친화적 디자인
- 밝고 친근한 색상과 이모지 사용
- 직관적인 화면 전환
- 명확한 버튼과 텍스트

### 교육적 가치
- 전략적 사고력 향상
- 자원 관리 능력 개발
- 공간 인지 능력 향상
- 경제적 사고 (코인 관리)

### 점진적 난이도
- 웨이브가 진행될수록 적의 체력 증가
- 스테이지가 진행될수록 난이도 증가
- 보스 등장으로 긴장감 조성

## 🚀 실행 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 홈 화면에서 "전투 시작" 클릭
3. 게임 영역을 클릭하여 타워 설치
4. 적들을 막아서 생명력을 보호하세요!
5. 스테이지 완료 후 상점에서 아이템 구매

## 🎨 기술 스택

- **HTML5 Canvas**: 게임 렌더링
- **JavaScript**: 게임 로직 및 상태 관리
- **CSS3**: 스타일링 및 애니메이션

## 🎮 게임 팁

1. **전략적 배치**: 경로의 곡선 부분에 타워를 설치하면 더 많은 적을 공격할 수 있습니다
2. **상점 활용**: 코인을 모아서 상점에서 아이템을 구매하여 능력을 강화하세요
3. **보스 대비**: 보스는 체력이 많으므로 여러 타워로 집중 공격하세요
4. **자원 관리**: 골드와 코인을 효율적으로 사용하여 최적의 전략을 세우세요
