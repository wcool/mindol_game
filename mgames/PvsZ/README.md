# 식물 vs 좀비 게임 🌻🧟

웹 기반 Plants vs Zombies 게임입니다.

## 게임 방법

### 시작하기
1. `index.html` 파일을 웹 브라우저에서 열어주세요.
2. 게임이 자동으로 시작됩니다.

### 게임 플레이
- **식물 배치**: 상단의 식물 카드를 클릭한 후, 게임 보드의 원하는 위치를 클릭하여 식물을 배치하세요.
- **태양 수집**: 화면에 떨어지는 태양(☀️)을 클릭하여 태양을 수집하세요.
- **좀비 방어**: 좀비가 왼쪽 끝에 도달하기 전에 모두 물리치세요!

### 식물 종류
- **🌻 해바라기 (50 태양)**: 10초마다 태양을 생성합니다.
- **🌱 완두콩 (100 태양)**: 좀비를 향해 완두콩을 발사합니다.
- **🥜 벽돌 (50 태양)**: 높은 체력을 가진 방어용 식물입니다.

### 게임 규칙
- 좀비가 왼쪽 끝에 도달하면 게임 오버입니다.
- 20마리 이상의 좀비를 물리치면 승리합니다!
- 좀비는 10초마다 랜덤한 행에 생성됩니다.
- 태양은 15초마다 랜덤한 위치에 생성됩니다.

## 파일 구조
- `index.html`: 게임 HTML 구조
- `style.css`: 게임 스타일
- `game.js`: 게임 로직

## 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla)
